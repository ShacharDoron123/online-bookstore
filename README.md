# 📚 Online Bookstore Web Application

A full-featured, responsive E-Commerce web application for browsing and purchasing books. Designed with a clean interface and smooth user experience.

## 🚀 Live Demo
👉 [Explore the Online Bookstore](https://shachardoron123.github.io/online-bookstore/)

---

## ✨ Key Features

- 📖 **Interactive Catalog:** Browse through a collection of books with visual promotional badges (*Best Seller*, *Summer Sale*, *Recommended*).
- 🏷️ **Dynamic Pricing & Sales:** Real-time visual representation of discounted prices vs. original prices.
- 🔍 **Detailed Book Views:** Dedicated pages for each book featuring full descriptions, author info, publication year, and technical details (pages, cover type).
- 🛒 **Full Shopping Cart System:**
  - Dynamic item quantity selection.
  - Interactive item removal.
  - Automated calculation of subtotal, applied discounts, and final total.
- 🧭 **Seamless Navigation:** Easy-to-use navigation header across all pages.
